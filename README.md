# Schutz – Marketplace de Ativos e Scripts

## Contexto do Projeto
O **Schutz** é uma plataforma web voltada para a comercialização de ativos digitais, scripts de automação e itens de jogos. O objetivo é conectar desenvolvedores e jogadores em um ambiente seguro, permitindo que usuários anunciem produtos, realizem compras via checkout integrado e avaliem a credibilidade dos vendedores. 

O sistema busca resolver a fragmentação do mercado de itens virtuais, oferecendo uma interface intuitiva e categorias organizadas por tipo de jogo ou utilitário.

## Tecnologias Previstas
*   **Front-end:** Next.js (React), Tailwind CSS e JavaScript.
*   **Back-end:** Node.js (API REST).
*   **Banco de Dados:** PostgreSQL.
